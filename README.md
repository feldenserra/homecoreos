------------------------------------------------------------

This is HomeCoreOS .

A framework to create your own efficient Home . 

------------------------------------------------------------

## How to Run
1. Clone this repo:
   `git clone https://github.com/feldenserra/homecoreos.git`
2. Modify the .env file to set pws .
3. Run with Docker:
   `docker-compose up -d --build`

The app will start at http://localhost:3000
